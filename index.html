Javascript Element Modification
Modify, Replace and Remove elements with Javascript by getting element id.

// This gets the element and changes it's display to 'none' (Does not appear but is in the code)
document.getElementById("element").style.display = "none";

// This changes the text in the element - The element here is 'hello'.
document.getElementById('hello').textContent = 'Hello, Console!'

// This writes "Hello JavaScript!" into an HTML element with id="demo". View the example at: http://bit.ly/fulldemohtml 
document.getElementById("demo").innerHTML = "Hello JavaScript!";
    
// To replace all existing classes with one or more new classes, set the className attribute.   
document.getElementById("MyElement").className = "MyClass";
